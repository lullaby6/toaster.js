# toaster.js

## Installation

```html
<link rel='stylesheet' href='https://cdn.jsdelivr.net/gh/lullaby6/toaster.js/toaster.css'>
<script src='https://cdn.jsdelivr.net/gh/lullaby6/toaster.js/toaster.js'></script>
```

## Usage

Call the function `Toaster` to invoke a toast

```js
Toaster({
    text: "Hello World!"
})
```
